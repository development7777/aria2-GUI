## About

aria2 GUIは、aria2をGUIで操作するためのツールです。QtのわかりやすいUIを提供します。


## 機能

- コマンドラインでできる機能をGUIで操作可能
- ダウンロードの追加、削除、停止、再開
- ダウンロードの進行状況の表示
- 設定の変更
- ログの表示
- Qtのシンプルで直感的なUI
